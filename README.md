#Little server with weather api

### how to run

* clone repositoty
* npm install
* npm run start

### queries

* /all (GET)
* /add (POST) requires a weather model 
* /edit (POST) requires a weather model
* /delete (POST) requires a weather model

### weather model

```
{
    "id" : 1,
    "city": "City name",
    "temp": 23
}
```
